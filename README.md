# An example of the implementation of a bottom drawer with an animated slider in the React Native App

# Getting Started

## Install modules:

```npm install```

### Install for iOS:

```
cd ios
pod install
```

## Start

```bash
# using npm
npm start

# OR using Yarn
yarn start
```
